E-Pharmacy Price Comparison System

A Python + Streamlit based web application that compares medicine prices across multiple online pharmacy and e-commerce platforms using SerpAPI Google Shopping results.


Features
Search medicines instantly
Compare prices from multiple platforms
Displays best/lowest price option
Interactive price comparison charts
Pie chart visualization
Direct product purchase links
Simple and responsive Streamlit UI
Technologies Used
Python
Streamlit
Pandas
Matplotlib
SerpAPI


Use Case

This project helps users quickly compare medicine prices online and identify the most affordable option from different sellers.
